**TuxGuitar** **Welcome to the new home of TuxGuitar, the free guitar tablature editor.** TuxGuitar was written by Julián Gabriel Casadesús and maintained by him for a very long time. At the end of 2022, the former TuxGuitar websites disappeared and the development in the [SourceForge repository](https://sourceforge.net/projects/tuxguitar/) stalled. After unsuccessful attempts to contact Julián, some TuxGuitar enthusiasts initiated a new [TuxGuitar repository](https://github.com/helge17/tuxguitar/) on Github. ![](snapshot.png) Download ============================================================== We provide ready to use installation packages for Linux, Windows, MacOS, FreeBSD and Android: * **[Latest stable version 1.6.2](https://github.com/helge17/tuxguitar/releases/tag/1.6.2)** * [Development snapshots and older releases](https://github.com/helge17/tuxguitar/releases) Help ============================================================== The online documentation of TuxGuitar is here: * **[Latest stable version 1.6.2](files/1.6.2/desktop/help/)** * [Development version](files/devel/desktop/help/) If you need help with TuxGuitar, you can ask a question on the [discussion](https://github.com/helge17/tuxguitar/discussions) page on Github. If you have found an error or would like to request a new feature, open a new [issue](https://github.com/helge17/tuxguitar/issues). Contribute ============================================================== If you would like to contribute to the further development of TuxGuitar, the instructions in the [doc](https://github.com/helge17/tuxguitar/blob/master/doc/) folder are a good starting point. Source Code ============================================================== The source code is available on [Github](https://github.com/helge17/tuxguitar/).
