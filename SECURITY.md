# Security Policy

## Reporting a Vulnerability

If you think you have found a security vulnerability in TuxGuitar, please create a private [vulnerability report](https://github.com/helge17/tuxguitar/security/advisories/new) on GitHub.

Alternativelly, you can send us an e-mail to [info@tuxguitar.app](mailto:info@tuxguitar.app).

When reporting a security vulnerability, please also follow the guidelines in the [ISSUES.md](docs/ISSUES.md) file.
